# LRU Project Management Installation Protocol
*Unified Core compatible install flow*

## Trigger
`install lru projects`

## Steps
1. Ask project type scope (coding/writing/research/business/all).
2. Create project folders under `projects/`.
3. Install project protocols/templates.
4. Update core docs references:
- add project command notes to `main/main-memory.md`
- add active project linkage in `main/current-session.md`
5. Verify create/load/save project cycle.

## Save Semantics Note
- `save project` persists project context only.
- `save` persists core memory and diary together.
