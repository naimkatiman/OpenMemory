# Time-based Aware System
*Temporal behavior extension for Unified Memory-Diary Core*

## What It Adds
- Time-aware greetings
- Time-based behavior adaptation
- Session timestamp context

## Integration Command
`Load time-aware-core`

## Unified Core Compatibility
This module now targets:
- `main/main-memory.md` for identity behavior updates
- `main/current-session.md` for active temporal context
- `main/session-format.md` for session field structure

It does not use split-memory files as active sources.
