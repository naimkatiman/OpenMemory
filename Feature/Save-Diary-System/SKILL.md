# Deprecated Save-Diary Skill Wrapper

This skill wrapper is retained for compatibility only.

1. Save-Diary feature is deprecated.
2. Use unified core `save` or `save diary` (same pipeline).
3. See `save-protocol.md` and `daily-diary/daily-diary-protocol.md`.
