# Recall Output Format
*Template for narrative recall responses*

## Single Match
- Natural confirmation
- Specific date reference
- Summary from diary evidence
- Link back to current conversation

## Multiple Matches
- Chronological summaries
- Pattern observation
- Natural continuation question

## No Match
- Explicit uncertainty
- Ask user for missing context

## Required Guardrails
- Never fabricate memories.
- Never output raw file paths to users.
- Always prioritize evidence from diary entries.
